# Umyjemy to za Ciebie

Profesjonalne mycie kostki brukowej, elewacji i dachów na terenie Łodzi i okolic.

## 🚀 Szybki Start

### Instalacja

```bash
npm install
```

### Uruchomienie lokalnie

```bash
npm start
```

Otwórz [http://localhost:3000](http://localhost:3000) w przeglądarce.

### Budowanie dla produkcji

```bash
npm run build
```

Tworzy folder `build` z zoptymalizowaną wersją do wdrożenia.

### Wdrażanie na GitHub Pages

```bash
npm run deploy
```

Ta komenda automatycznie zbuduje projekt i wdrożyć go na GitHub Pages.

## 📋 Wymagania

- Node.js (wersja 14 lub nowsza)
- npm lub yarn

## 🛠 Technologie

- React 18
- Tailwind CSS
